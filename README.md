Lumio is an adaptive, multi-sensory learning platform designed to support students with learning disabilities.

Instead of forcing students to fit into traditional education systems, Lumio adapts learning to how each student thinks, processes, and understands information.

---

## The Idea

Traditional learning assumes one way of understanding.

Lumio does the opposite.

It removes barriers that hide a student’s real intelligence and replaces them with tools that allow them to:
- understand better  
- express themselves freely  
- learn without pressure  

---

## Core Philosophy

> The student must never feel like the system is waiting for them to fail.

- Every pause is met with support  
- Every mistake is redirected  
- Every session ends with proof that the student is capable

## Features

### Dyslexia Module (Phase 1)

#### Visual Learning from Text

- Capture homework using camera  
- Extract and understand text  
- Convert content into generated visuals (images)  
- Optional AR overlay support (WebXR ready)  

Example:  
A story like *The Hare and the Tortoise* becomes visual scenes instead of just text.

---

#### Text to Speech + Reading Support

- Reads original text aloud  
- Reads simplified explanations  
- Word by word highlighting  
- Reading ruler for guided focus  

---

#### Voice to Text (Core Feature)

- Speak answers naturally  
- Get a clean written version  
- Listen back to responses  
- Edit without pressure  

**Impact:**
- separates thinking from writing  
- reduces cognitive load  
- reveals true understanding  

---

#### Writing Support

- gentle spelling feedback  
- no overwhelming error indicators  
- preserves natural voice  
- encourages refinement instead of punishment  

---

#### Intelligent Reading Assistance

Lumio detects:
- long pauses  
- repeated audio playback  
- abandoned attempts  
- inactivity  

And responds by:
- slowing down delivery  
- breaking content into smaller parts  
- offering alternative explanations  
- guiding calmly without pressure

## How to Run

### Prerequisites
- Node.js installed on your computer (https://nodejs.org)

### Installation

1. Clone the repository:
   git clone https://github.com/modisapudikamogelo78-coder/Lumio.git

2. Navigate into the project folder:
   cd Lumio

3. Install dependencies:
   npm install

4. Start the development server:
   npm run dev

5. Open your browser and go to:
   http://localhost:5173
